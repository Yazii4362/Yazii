## What I'm Good At
 
- **기반을 무너지지 않게 쌓기**
  컬러시스템, 파일 구조, 브랜드 아이덴티티 — "
  왜 이렇게 설계해야 하는지 납득하고 넘어갑니다.
  느려 보이지만, 나중에 흔들리지 않는 결과물을 만드는 방식입니다.
 
- **브랜드를 "느낌"이 아니라 "논리"로 보기**
  예쁜 걸 고르기 전에 이 브랜드가 어떤 사람에게 어떻게 보여야 하는지를 먼저 설정합니다.
  UX와 브랜딩은 연결된다고 생각합니다.
 
- **추상적인 아이디어를 시스템으로 정리하기**
  아이디어가 넘치되, 구조 없이 흘려보내지 않습니다.
  개념을 잡으면 실제 화면과 구조로 옮겨보며 실험합니다.
 
- **초반 몰입으로 후반 수정 줄이기**
